# Descripció del Projecte: Incubadora IT

Durant quatre setmanes la classe funcionarà com una incubadora d’empreses IT. L’objectiu és dissenyar una startup tecnològica i defensar-la en una presentació final davant un comitè simulant una ronda d’inversió.

No es tracta només de generar una idea, sinó de construir una proposta completa, viable i argumentada.

## Objectius del Projecte

El projecte té dos objectius principals:

### 1. Construir una empresa IT
L’equip haurà de:
- Detectar un problema real
- Identificar un client real
- Dissenyar una solució tecnològica
- Definir un model de negoci
- Analitzar costos, riscos i viabilitat
- Desenvolupar un prototip funcional
- Preparar una presentació tipus pitch

### 2. Construir el perfil professional
Cada alumne haurà de:
- Definir el seu rol dins de l’equip
- Identificar les seves fortaleses i valor professional
- Elaborar un pla de projecció laboral

## Fases del Projecte

El projecte es divideix en quatre fases, una per setmana:

**Setmana 1 — Problema + Client**  
Identificació d’un problema real i del client que el pateix.

**Setmana 2 — Estratègia + Empresa**  
Definició de la proposta de valor, posicionament i model de negoci.

**Setmana 3 — Prototip + Viabilitat**  
Construcció d’un prototip i estudi de costos, preus, marge i riscos.

**Setmana 4 — Validació + Pitch**  
Refinament, validació i preparació de la presentació final.

## Producte Final

Al final del projecte l’equip haurà de lliurar:

- Una empresa IT definida
- Un prototip funcional
- Un model de negoci coherent
- Una anàlisi de viabilitat
- Un pitch final davant d’un comitè

## Resultat Esperat

L’alumnat haurà demostrat:
- Capacitat d’anàlisi i resolució de problemes
- Pensament estratègic
- Treball en equip
- Competència tècnica
- Comunicació i defensa de propostes

El projecte té com a finalitat aproximar l’alumnat al context professional i generar evidències del seu valor com a futurs tècnics.
